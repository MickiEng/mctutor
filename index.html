CSV File Requirements

Your CSV file requires a header row with the column names below. Only the Required Columns must have data for every question.

Required Columns (Must contain data)

  • id: A unique ID for the question (e.g., Q001).
  • stem: The question text itself.
  • choice_a, choice_b, choice_c, choice_d: The four answer options.
  • correct: The correct answer label (A, B, C, or D).

Optional Columns (Can be left blank)

  • topic: The subject or topic (e.g., Fundamentals, OB).
  • bloom: Bloom's taxonomy level (e.g., Remember, Analyze).
  • choice_a_rationale, choice_b_rationale, choice_c_rationale, choice_d_rationale: Explanations for why each choice is correct or incorrect.
  • embed_code: HTML embed code (e.g., for relevant YouTube, images) to show after answering.
